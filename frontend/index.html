关于不认命 App

不认命 App 是一款基于传统八字命理学的寺庙匹配与祈福指南应用。 通过分析您的生辰八字，为您推荐最适合的寺庙和祈福方向。

功能特色

  • ✅ 八字计算 - 精准计算生辰八字
  • ✅ 五行分析 - 分析五行强弱
  • ✅ 喜用神推算 - 找出最弱的五行
  • ✅ 寺庙匹配 - 推荐最适合的寺庙
  • ✅ 万年历 - 阳历农历转换
  • ✅ 10,857 座寺庙数据库

技术栈

后端：FastAPI + Python
算法：八字计算 + 寺庙匹配
数据：10,857 座寺庙（47.6% 有历史介绍）
