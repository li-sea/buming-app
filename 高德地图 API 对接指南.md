## 📋 第一步：注册高德开发者（5 分钟）

### 1. 访问高德开放平台

- 点击右上角 **"登录/注册"**

### 2. 注册账号
- 使用手机号或邮箱注册
- 完成实名认证（需要身份证）

### 3. 创建应用
1. 进入 **"控制台"** → **"应用管理"**
2. 点击 **"创建新应用"**
3. 填写信息：
   - **应用名称:** 不认命 App
   - **应用类型:** 其他
   - **描述:** 寺庙文化数据收集

## 🔧 常见问题

### Q1: 提示 "INVALID_USER_KEY"
**A:** API Key 配置错误，检查是否复制完整

### Q2: 提示 "DAILY_QUERY_OVER_LIMIT"
**A:** 超出每日限额，等明天或申请提额

### Q3: 数据量少
**A:** 
- 检查关键词是否准确
- 增加城市列表
- 增加页数（`range(1, 4)` 改为 `range(1, 6)`）

### Q4: 脚本运行慢
**A:** 
- 正常现象（控制频率避免限流）
- 可适当减少 `time.sleep()` 时间
- 或减少城市/关键词数量
