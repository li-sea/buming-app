## 📱 人工测试指引

### 测试步骤

#### 1. 页面加载测试
- [ ] 打开 http://localhost:8080
- [ ] 检查页面是否正常加载
- [ ] 检查标题是否为"不认命 🔮 - 传统文化学习工具"
- [ ] 检查免责声明是否显示在页面顶部
- [ ] 检查按钮文字是否为"🔮 开始探索"

#### 2. 输入表单测试
- [ ] 填写出生年份（1900-2100）
- [ ] 填写出生月份（1-12）
- [ ] 填写出生日期（1-31）
- [ ] 填写出生时辰（0-23）
- [ ] 选择性别（男/女）
- [ ] 选择祈福方向（事业/财运/姻缘/健康/平安/学业/修行）
- [ ] 填写所在地区（可选）

#### 3. 结果展示测试
- [ ] 点击"开始探索"按钮
- [ ] 检查加载动画是否正常
- [ ] 检查八字结果是否正确显示
- [ ] 检查五行分布图表是否正常
- [ ] 检查推荐寺庙列表（最多 5 座）
- [ ] 检查"文化契合度"文案（非"匹配度"）
- [ ] 检查法器推荐（文化含义，无价格）

#### 4. 合规检查
- [ ] 所有文案是否使用"传统文化"定位
- [ ] 是否出现"算命""预测""改运"等禁止用语
- [ ] 免责声明是否显著
- [ ] 底部是否有用户协议/隐私政策链接
- [ ] 法器推荐是否无价格信息

#### 5. 移动端适配测试
- [ ] 用手机浏览器访问 http://localhost:8080
- [ ] 检查页面是否自适应屏幕
- [ ] 检查表单输入是否方便
- [ ] 检查结果展示是否清晰

#### 6. 性能测试
- [ ] 页面加载时间 < 3 秒
- [ ] API 响应时间 < 1 秒
- [ ] 连续点击多次无报错
