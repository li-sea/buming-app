# 🌳 不认命 App - 祈福树功能

**创建时间:** 2026-04-27  
**版本:** v1.0.0

---

## 📋 功能概述

祈福树是不认命 App 的社交功能，用户可以：
1. **种一棵祈福树** - 选择祈福方向，写下心愿
2. **邀请好友点赞** - 分享链接，邀请好友点赞
3. **树成长系统** - 点赞越多，树长得越高
4. **祈福广场** - 浏览别人的树，随喜点赞

## 🔌 API 文档

### 1. 种一棵祈福树

**请求:**
```
POST /api/pray-tree/create
Content-Type: application/json

{
    "user_id": "user_123",
    "prayer_focus": "事业",
    "wish": "事业顺利，升职加薪",
    "birth_date": "1990-03-15"  // 可选，用于星空图
}
```

**响应:**
```json
{
    "status": "success",
    "tree": {
        "tree_id": "tree_1234567890_1234",
        "user_id": "user_123",
        "prayer_focus": "事业",
        "prayer_emoji": "💼",
        "wish": "事业顺利，升职加薪",
        "level": 1,
        "likes": 0,
        "status": "growing",
        "tree_emoji": "🌱"
    }
}
```

### 2. 给祈福树点赞

**请求:**
```
POST /api/pray-tree/like
Content-Type: application/json

{
    "tree_id": "tree_1234567890_1234",
    "liker_id": "liker_456"
}
```

**响应:**
```json
{
    "status": "success",
    "tree": {
        "tree_id": "tree_1234567890_1234",
        "likes": 1,
        "level": 2,
        "status": "growing"
    }
}
```

### 3. 获取祈福树信息

**请求:**
```
GET /api/pray-tree/{tree_id}
```

**响应:**
```json
{
    "status": "success",
    "tree": {
        "tree_id": "tree_1234567890_1234",
        "user_id": "user_123",
        "prayer_focus": "事业",
        "wish": "事业顺利，升职加薪",
        "level": 2,
        "likes": 1,
        "status": "growing",
        "tree_emoji": "🌱"
    }
}
```

### 4. 获取用户所有祈福树

**请求:**
```
GET /api/pray-tree/user/{user_id}
```

**响应:**
```json
{
    "status": "success",
    "trees": [
        {
            "tree_id": "tree_1234567890_1234",
            "prayer_focus": "事业",
            "wish": "事业顺利",
            "level": 2,
            "likes": 1
        }
    ]
}
```

### 5. 获取热门祈福树

**请求:**
```
GET /api/pray-tree/hot?limit=20
```

**响应:**
```json
{
    "status": "success",
    "trees": [
        {
            "tree_id": "tree_1234567890_1234",
            "prayer_focus": "事业",
            "likes": 10,
            "level": 10
        }
    ]
}
```

### 6. 获取随机祈福树（祈福广场）

**请求:**
```
GET /api/pray-tree/random?limit=20
```

**响应:**
```json
{
    "status": "success",
    "trees": [
        {
            "tree_id": "tree_1234567890_1234",
            "prayer_focus": "事业",
            "likes": 5,
            "level": 6
        }
    ]
}
```

### 7. 分享祈福树

**请求:**
```
POST /api/pray-tree/{tree_id}/share
```

**响应:**
```json
{
    "status": "success",
    "share": {
        "tree_id": "tree_1234567890_1234",
        "share_count": 1,
        "share_url": "https://buming.app/pray/tree_1234567890_1234"
    }
}
```
